# flattree

Port of @mafintosh's flat-tree functions to map a binary tree to a list.

## Prerequisites

* [Kotlin/Native](https://github.com/JetBrains/kotlin-native) and the
  `konanc` command line program.

## Usage

### Compile klib:
```sh
$ make library
```

### Run tests:
```sh
$ make test && ./test.kexe
```
